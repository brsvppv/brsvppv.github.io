Personal Use Integration Policy

Last Updated: June 2024

1. Integration Overview

This personal integration connects Home Assistant with Google Assistant for:

  • Voice control of smart home devices
  • State synchronization between systems
  • Household automation management

Exclusively for private use with no external access or distribution.

2. Data Collection

Collected Data

  • Device states (on/off, temperature, etc.) from Home Assistant
  • Google Assistant voice command transcripts
  • OAuth tokens for authentication between systems
  • No personal data collected beyond what's required for functionality

3. Data Usage

How We Use Data

  • Enable voice control through Google Assistant
  • Maintain device state consistency
  • Provide status feedback in Google Home app

No AI Training

No Advertising

No Third-Party Sharing

4. Data Retention

Google Servers

Automatically deleted after 10 days (Google policy requirement)

Home Assistant

Retained only during active sessions (<1 day)

No persistent storage of Google-related data

User Control

Revoke via Google Account

5. Security Measures

Encryption

  • HTTPS for all communications
  • OAuth 2.0 authentication

Access Controls

  • 4-digit PIN for security devices
  • Firewall-protected local network
  • No inbound internet connections

6. Compliance

This integration complies with:

  • Google Home Developer Policies
  • Home Assistant Cloud Terms
